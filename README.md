
# Límites de la Predicción Sincrónica (v2)

## Resumen

Teoría unificada, experimental y falsable que introduce la **Acción Sincrónica** como magnitud central para cuantificar los límites fundamentales de la predicción.

La accesibilidad predictiva está limitada por **cuatro mecanismos**:
- Dinámico (Caos)
- Computacional
- Inferencial (Modelo)
- Físico

## Conceptos clave

- **Accesibilidad predictiva**  
  `A(h) = 1 - MSE_modelo / MSE_baseline`

- **Horizonte predictivo**  
  `h* = min { h : A(h) < ε }`

- **Acción Sincrónica**  
  `S(R) = ∫₀^{H_físico} A(h, R) dh`

- **Ley de Límite Unificado**  
  `A(h, R) ≤ min(A_din, A_comp, A_inf, A_fis)`
